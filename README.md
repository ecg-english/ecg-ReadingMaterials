# ECG Reading Materials

英会話ジムECGの英語学習者向けリーディング教材プロジェクトです。

## 概要

シェイクスピアの作品をはじめ、様々な英語文学作品を教材として提供します。各教材には以下の5段階の構成が含まれています：

1. **英語原文** - オリジナルのテキスト
2. **和訳** - 日本語訳
3. **単語解説** - 重要単語の詳細な解説
4. **文法解説** - 文法ポイントの説明
5. **作品解説** - 歴史的背景、著者情報、文学的解説

## 特徴

✨ **インタラクティブなハイライト機能**
- 単語や文法の解説をクリックすると、本文中の該当箇所が金色にハイライト表示されます
- マウスホバーでプレビュー表示も可能

📖 **本物の書籍のようなデザイン**
- 古い紙のような質感とテクスチャ
- インクペンのようなエレガントなセリフフォント
- 3D効果とアニメーション

## 使用方法

1. ブラウザで`index.html`を開く
2. 単語や文法の解説カードをクリックして、本文中の該当箇所を確認
3. マウスをホバーすると、軽いプレビューハイライトが表示されます

## 技術スタック

- HTML5
- CSS3 (カスタムアニメーション、グラデーション、3D効果)
- Vanilla JavaScript (インタラクティブ機能)
